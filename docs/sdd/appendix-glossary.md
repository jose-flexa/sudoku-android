# Appendix - Glossary

| Term | Definition |
| --- | --- |
| Sudoku | Number-placement puzzle on a 9x9 grid with row/column/subgrid constraints |
| Clue Cell | Pre-filled cell that cannot be edited by player |
| Candidate Notes | Temporary possible values stored by player in a cell |
| Unique Solution | Puzzle has exactly one valid solved board |
| MVVM | Model-View-ViewModel architectural pattern |
| Compose | Android declarative UI toolkit |
| Room | Android ORM over SQLite for local persistence |
| DataStore | Jetpack key-value/preference storage API |
| Use Case | Encapsulated domain action representing business logic |
| StateFlow | Kotlin Flow type for observable state streams |
